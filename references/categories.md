# Categorias Shopee

Esta página lista os IDs de categorias disponíveis na API de Afiliados da Shopee.

## Como obter a lista completa

A lista completa de categorias pode ser obtida através do:

1. **API Explorer**: [https://open-api.affiliate.shopee.com.br/explorer](https://open-api.affiliate.shopee.com.br/explorer)
   - Navegue para `productOfferV2`
   - O campo `productCatId` mostra as categorias disponíveis

2. **Query de produtos**: Use a query `productOfferV2` sem filtro de categoria para ver os `productCatIds` retornados nos produtos
