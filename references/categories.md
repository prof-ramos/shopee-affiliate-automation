## Como Usar

### Python

```python
# Buscar produtos de Eletrônicos
result = shopee.search_products(
    category_id=10001,
    list_type=1,
    sort_type=5,
    limit=20
)

# Buscar smartphones especificamente
result = shopee.search_products(
    category_id=10002,
    list_type=1,
    sort_type=5,
    limit=50
)
```

### GraphQL Direct

```graphql
query {
  productOfferV2(
    productCatId: 10002
    listType: 1
    sortType: 5
    page: 1
    limit: 20
  ) {
    nodes {
      productName
      commissionRate
      priceMin
      offerLink
    }
  }
}
```

### N8N Function Node

```javascript
// Buscar em múltiplas categorias
const categories = [10002, 10003, 10004];  // Tech categories

const queries = categories.map(catId => ({
  query: `
    query {
      productOfferV2(
        productCatId: ${catId}
        listType: 1
        sortType: 5
        page: 1
        limit: 10
      ) {
        nodes {
          productName
          commissionRate
          offerLink
        }
      }
    }
  `
}));

return queries.map(q => ({ json: q }));
```

## Categorias Mais Rentáveis

Baseado em comissões médias (pode variar):

### Alta Comissão (>10%)

- Eletrônicos (10001)
- Celulares e Tablets (10002)
- Beleza e Maquiagem (40001-40002)
- Suplementos (60002, 80005)

### Média Comissão (5-10%)

- Moda (20001-20002)
- Casa e Decoração (30001-30007)
- Esportes (50001-50007)

### Variável

- Alimentos (80001-80005) - geralmente baixa
- Livros (90001-90002) - geralmente baixa
- Pet Shop (95001-95005) - varia muito

## Dicas de Busca por Categoria

### Combinar Categoria + Palavra-chave

```python
# Buscar "fone bluetooth" na categoria Áudio
result = shopee.search_products(
    keyword="bluetooth",
    category_id=10005,
    list_type=1,
    sort_type=5,
    limit=30
)
```

### Filtrar por Comissão Mínima

```python
# Buscar produtos de beleza com comissão > 8%
result = shopee.search_products(
    category_id=40001,
    list_type=1,
    sort_type=2,  # Ordenar por comissão
    limit=50
)

products = result['data']['productOfferV2']['nodes']
high_commission = [p for p in products if p['commissionRate'] > 8.0]
```

### Comparar Múltiplas Categorias

```python
import asyncio

async def compare_categories(cat_ids):
    tasks = []
    for cat_id in cat_ids:
        task = shopee.search_products(
            category_id=cat_id,
            list_type=1,
            sort_type=5,
            limit=10
        )
        tasks.append(task)
    
    results = await asyncio.gather(*tasks)
    return results

# Comparar eletrônicos vs moda vs beleza
results = asyncio.run(compare_categories([10001, 20001, 40001]))
```

## Notas Importantes

- IDs são aproximados e podem variar
- Shopee pode adicionar/remover categorias
- Teste sempre com API Explorer antes de implementar
- Algumas categorias podem ter subcategorias não listadas
- Use `sortType: 5` (mais vendidos) para melhores resultados
