# Deploy

Este repositorio contem documentacao e recursos de uma skill Claude. Nao ha
pipeline de build ou deploy.

## Distribuicao

- Empacote o conteudo do repositorio como uma pasta de skill.
- Garanta que `SKILL.md` e `references/` estejam incluidos.
- Se scripts forem adicionados, inclua `scripts/` e atualize as docs.
